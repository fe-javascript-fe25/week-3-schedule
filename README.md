# Schema, vecka 3
###### JavaScript, vecka 2 av 8

## Introduktion

Under den andra veckan fortsätter vi att bygga vidare på våra grundläggande kunskaper i JavaScript. Fokus ligger på objekt och arrayer – två centrala datastrukturer som gör det möjligt att organisera, lagra och hantera information på ett mer strukturerat och effektivt sätt. Vi kommer att utforska hur de fungerar, hur de används i praktiken och varför de är så viktiga när vi ska skriva mer avancerad och skalbar kod framöver. Genom konkreta exempel och övningar får du en bättre förståelse för hur objekt och arrayer samverkar med variabler, funktioner och loopar i verkliga programmeringssituationer.


## Mål för veckan:

1. Förstå vad ett objekt är, samt hur det används
2. Förstå vad en array är, samt vilka dess mest grundläggande funktioner är och hur de fungerar
